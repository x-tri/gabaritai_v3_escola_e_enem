## Como Capturar

### macOS
- `Cmd + Shift + 4` - Selecionar area
- `Cmd + Shift + 4 + Space` - Capturar janela

### Windows
- `Win + Shift + S` - Ferramenta de recorte
- `Alt + PrintScreen` - Janela ativa

## Recomendacoes

1. Use resolucao 1920x1080 ou superior
2. Capture com tema claro para melhor visualizacao
3. Remova informacoes sensiveis (nomes reais de alunos)
4. Mantenha proporcao 16:9 quando possivel
